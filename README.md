# Webpack starter
webpack starter for me to reuse

Includes:
<li>Axios</li>
<li>Babel loader</li>
<li>Bootstrap</li>
<li>Bootstrap icons</li>
<li>Dotenv</li>
<li>HTMLWebpackPlugin</li>
<li>Lodash</li>
<li>Sass loader</li>
<li>Source Maps</li>
<li>WebpackDevServer</li>

## Usage
Run Dev Server (Port: 3000)
npm run dev
  
## Build for production
npm run build
